# Simple desk calculator.

Coming from http://caml.inria.fr/pub/docs/manual-ocaml-4.00/manual026.html.

## Goal

Learn and try ocamllex and menhir.

## How to use?

Compile and run:
```
make && ./desk_calculator
```

Some examples (lines beginning with `>` is the output):
```
5 + 6
> 11
11 * 3
> 33
(9 + 9) * 3
> 54
```

Clean:
```
make clean
```
